# Accent Swap
A snippet that makes it simple to swap the Accent/Brand color of Discord

### Importing:
#### Import
```css
@import url(//OILYY.github.io/css-snippets/AccentSwap/import.css);
:root:root {
    --accent-h: 235;
    --accent-s: 86%;
    --accent-l: 65%;
}
```
